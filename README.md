# Processing API Platform + Analytics Dashboard

## Overview
This is a full-stack SaaS platform for processing data via an API and monitoring usage through a dashboard. The project demonstrates API design, authentication, rate limiting, analytics, and full-stack development using modern technologies.

## Tech Stack
- **Backend:** Java 21, Spring Boot 4, H2 Database (initial setup), PostgreSQL (future), Redis
- **Frontend:** React, Tailwind CSS
- **Infrastructure:** Docker, CI/CD with GitHub Actions

## Project Structure
- `/backend` - Spring Boot API
- `/frontend` - React dashboard (coming soon)

## Current Status
- H2 console set up for initial development
- API and dashboard not yet implemented
